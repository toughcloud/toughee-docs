接入设备查询

NAS 查询

接口描述

第三方系统向硬派计费系统发起NAS查询请求，硬派计费返回NAS信息列表。支持可选的NAS IP地址参数。

NAS 数据结构：

接口地址

http://服务器地址:端口/api/v1/nas/query

请求参数

  • ip_addr：nas ip地址

CURL 请求示例：

curl "http://127.0.0.1:1816/api/v1/nas/query?sign=C746B12E9B57BD2EBF0FDF78F7DFFABE"
